## 本家様
[https://github.com/setchi/NoteEditor](https://github.com/setchi/NoteEditor)

## 本家様からの変更内容
### システム
SEの音量調整追加
### 譜面
カテゴリ追加（自作ゲーム用）  
難易度追加  
難易度レベル追加
### ノーツ
フリック（8方向）ノーツの追加
### 実装中
カーブロング（左右）ノーツの追加  
ノーツサイズの追加  
汎用的に使用できる特殊ノーツの追加
Offset追加
